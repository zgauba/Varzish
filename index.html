Program Strategy

Why This Works For You

GLP-1 microdosing suppresses appetite and aids fat oxidation — but it also risks muscle loss if protein intake and resistance training are insufficient. This program counteracts that directly: progressive calisthenics preserves and builds lean tissue while the caloric deficit burns the belly fat. Muscle definition appears as the fat comes off — these two goals work together, not against each other.

Daily Time Commitment

  • Strength/Full-body days: 30 min
  • Mobility days: 25 min
  • Active recovery: 20 min walk
  • Rest day: Full rest

Phase Progression (12 Weeks)

  • Phase 1 (Wks 1–4): Foundation — form, habit, baseline strength
  • Phase 2 (Wks 5–8): Build — add pulling, unilateral legs, more volume
  • Phase 3 (Wks 9–12): Peak — harder variations, longer holds, full definition

What Changed in v2

  • Added Door Towel Row — fixes the missing pulling movement for back definition
  • Added Reverse Lunge — unilateral leg work for better definition in Phase 2+
  • Added Glute Bridge March — progression from standard bridge in Phase 2+
  • Removed Wall Push-Up — replaced by Knee Push-Up as the true Phase 1 entry
  • All workouts capped at 30 min — realistic for a desk worker
  • Custom illustrations — uniform muscle-highlight images for every exercise

Desk Warrior Fix

The Problem

What Computer Work Does

  • Tight hip flexors — from constant sitting, pulls lumbar into anterior tilt
  • Weak glutes — inhibited from sitting, causes lower back pain
  • Forward head posture — every inch forward = +10 lbs load on cervical spine
  • Rounded shoulders — pec tightness, weak mid-back (rhomboids/lower traps)
  • Thoracic kyphosis — upper back rounds, reduces breathing capacity

Ergonomic Setup

Quick Desk Wins

  • Monitor height: Top of screen at eye level
  • Chair height: Feet flat on floor, knees at 90°
  • Lumbar support: Small pillow or rolled towel at lower back
  • Screen distance: Arm's length away
  • Set a timer: Get up every 60 min. Walk 2 min. Non-negotiable.
